Install
--------


Install from source
~~~~
.. code-block::

    git clone https://github.com/ChengFR/polyphony.git
    make install

If you want to install the package for development. Please run:

.. code-block::

    make install-develop

How to use
--------
Start the server by

.. code-block::

    polyphony
